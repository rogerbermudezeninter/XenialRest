# Imágenes de productos

Coloca aquí las imágenes de los productos. La ruta que configures en cada producto debe ser relativa a esta carpeta.

**Ejemplo:**
- Archivo: `imagenes/productos/ensalada.jpg`
- Ruta en ConfigPanel → Productos → Imagen: `productos/ensalada.jpg`

La API sirve las imágenes en `http://localhost:3000/imagenes/`.
